# Todo List ReactJS

A simple and elegant todo list application built with React.

## Features

- Add, edit, and delete todos
- Mark todos as complete
- Clean and user-friendly interface
- Responsive design

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/zyadcs50/Todo-List-ReactJS.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Usage

- Enter a new todo in the input field and press Enter to add it
- Click the checkbox to mark a todo as complete
- Click the delete button to remove a todo
